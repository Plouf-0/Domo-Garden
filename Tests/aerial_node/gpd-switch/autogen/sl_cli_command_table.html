node

gpd node related commands
node info sl_zigbee_gpd_af_cli_node_info
Prints the GPD node information.
node channel-req sl_zigbee_gpd_af_cli_channel_req
Sends out a channel request command from the GPD.
node comm u8 sl_zigbee_gpd_af_cli_comm_req
Sends out a commissioning request and moves the commissioning state machine based on the parameter provided.
Arguments
  • u80x00 - Decommission, 0x01 = A single commissioning GPDF, 0xFF = Start automatic commissioning process
node set-addr hex sl_zigbee_gpd_af_cli_node_set_addr
Sets the address of the GPD node.
Arguments
  • hexarray of 8 bit data, where AppId = arg[0], if AppId = 0 then SrcId = arg[1-4] but if AppId = 2 then Endpoint = arg[1] and EUI = arg[2-9]
node set-ac u8 sl_zigbee_gpd_af_cli_node_set_auto_comm
Sets the auto-commission parameter of the GPD node.
Arguments
  • u8Auto-commissioning parameter of the GPD node
node set-bidir-param u8 u8 u8 sl_zigbee_gpd_af_cli_node_set_bi_dir_param
Sets bidirectional parameter of the GPD node.
Arguments
  • u8rx-after-tx flag.
  • u8rx-offset in milliseconds.
  • u8rxwindow in milliseconds.
node set-channel u8 sl_zigbee_gpd_af_cli_node_set_channel
Sets the channel of operation for unidirectional GPD or channel for bidirectional commissioning.
Arguments
  • u8GPD channel from 11 to 26
node set-sec-key-type u8 sl_zigbee_gpd_af_cli_node_set_sec_key_type
Sets the bidirectional commissioning key type.
Arguments
  • u8Key type between 0 and 7
node set-sec-key hex sl_zigbee_gpd_af_cli_node_set_sec_key
Sets the bidirectional key.
Arguments
  • hex128 bit key
node set-sec-fc u32 sl_zigbee_gpd_af_cli_node_set_sec_fc
Sets security frame counter.
Arguments
  • u32Sec frame counter
node set-sec-level u8 sl_zigbee_gpd_af_cli_node_set_sec_level
Sets security level. 0 = No security, 1 = RESERVED, 2 = Authentication only, 3 = Encrypted and Authenticated
Arguments
  • u8Sec level between 0 to 3.
